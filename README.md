# mouse-lfp-transformer

## Overview

University project applying deep learning to neural decoding of visual stimuli
from mice Local Field Potential (LFP) recordings. Data comes from the
[Allen Brain Observatory — Visual Coding (Neuropixels)](https://portal.brain-map.org/explore/circuits/visual-coding-neuropixels)
dataset. The model classifies 8 types of visual stimuli from time-binned
bandpower features extracted from Neuropixels LFP signals recorded in the
primary visual cortex (VISp) of 7 mouse sessions.

**Best test accuracy: 79.9% (8-way classification, 28,814 held-out trials)**

---

## Methodology

The core model combines a **Transformer encoder** (for cross-time-bin attention)
with a **GRU** (for sequential dynamics), operating on 10 time-binned bandpower
features per trial. Input shape: `(B, K=10, D=532)` — 10 bins × 76 channels ×
7 frequency bands.

```
Input (B, 10, 532)
  → Linear projection 532→128 + LayerNorm
  → TransformerEncoder (1 layer, 4 heads)
  → GRU (2 layers, hidden=128)
  → LayerNorm → Dropout → Linear(128→64) → ReLU → Dropout → Linear(64→8)
  → Logits (B, 8)
```

A linear regression baseline is included for comparison.
Class imbalance is handled via per-class weighted cross-entropy loss.

## Usage

### 1. Download and preprocess data

Run notebooks in order (requires `allensdk`, several GB of disk space):

```
notebooks/01_data_retrieval.ipynb
notebooks/02_downsampling.ipynb
notebooks/03_feature_extraction.ipynb
```

> The raw `.npy` / `.pt` files are excluded via `.gitignore`.
> Re-run preprocessing notebooks to reproduce features from scratch.

### 2. Train models

```bash
# GRU-Transformer (main model)
python scripts/train_gru_transformer.py --config config.yaml --data-dir /path/to/data

# Linear baseline
python scripts/train_linear_baseline.py --config config.yaml --data-dir /path/to/data --model mlp

# Dry-run (validate config without training)
python scripts/train_gru_transformer.py --config config.yaml --dry-run
```

On GPU (CUDA), 100 epochs take approximately 1 hour.

### 3. HPC (BU SCC)

```bash
qsub scripts/run_hpc.sh
```

### 4. Reproduce F1 figure

After training, run `notebooks/05_results_f1_plot.ipynb`.
Output saved to `results/f1_per_class.pdf`.

### Configuration

All hyperparameters live in `config.yaml`. Key settings:

```yaml
model:
  d_model: 128
  n_heads: 4
  n_transformer_layers: 1
  n_gru_layers: 2
  dropout: 0.10

training:
  num_epochs: 100
  learning_rate: 3.0e-4
  batch_size: 64
  label_smoothing: 0.05
```

|---|---|
| Linear regression (flat features) | ~65% |
| **GRU-Transformer (time-series)** | **79.9%** |

| Stimulus | Accuracy |
|---|---|
| natural_movie_one_shuffled | 91.7% |
| gabors | 88.5% |
| natural_movie_one_more_repeats | 78.5% |
| drifting_gratings_75_repeats | 72.7% |
| spontaneous | 70.3% |
| flashes | 63.2% |
| dot_motion | 61.0% |
| drifting_gratings_contrast | 53.2% |
